# Norwest Fresh Seafood — Website Development Project

> SPC3039 External Project · Group D · Academy of Interactive Technology · 2026

## 🐟 About the Project

This repository contains the source code for the Norwest Fresh Seafood website, developed as part of the SPC3039 External Project unit at AIT.

Norwest Fresh Seafood is a seafood retail business located at Norwest Marketplace, 4 Century Circuit, Norwest NSW 2153, run by Peter Noutaros. The store sells fresh seafood, gourmet products, cooked food, frozen seafood and offers catering for restaurants, clubs and events.

🏆 2022 Sydney Hills Local Business Award — Outstanding Fresh Food
